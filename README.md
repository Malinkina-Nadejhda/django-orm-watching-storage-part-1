# main.py

Скрипт для работы с базой данных пропусков


**Как установить:**

- Клонируйте репозиторий
- Должен быть установлен `Python` версии 3.11 или выше
- Используйте `pip` для установки зависимостей:

```bash
pip install -r requirements.txt
```

## Настройка окружения

1. Создайте файл `.env` в корневой папке проекта.
2. Добавьте в `.env` реальные данные для подключения:
   ```env
   DB_NAME=имя_вашей_базы
   DB_USER=ваш_логин
   DB_PASSWORD=ваш_пароль
   DB_HOST=адрес_хоста
   DB_PORT=5432
   ```

## Запуск и режимы работы

Используйте аргумент `--mode`. Скрипт поддерживает три режима работы. Вы можете использовать флаг `--help` для вывода подробной справки по аргументам.

### 1. Вывод общего количества пропусков (`count`)

Скрипт быстро посчитает и выведет общее число пропусков, зарегистрированных в базе данных. Этот режим работает по умолчанию, если запустить программу без флагов.

```bash
python main.py --mode count
```

![img](https://i.ibb.co/hxnqMZyT/13.png)


### 2. Вывод количества активных пропусков (`active`)

Использует эффективную фильтрацию для подсчета только действующих пропусков работающих сотрудников.

```bash
python main.py --mode active
```

![img](https://i.ibb.co/RkW9yTrK/14.png)


### 3. Информация о случайном пропуске (`info`)

Выбирает случайный пропуск из базы данных и  выводит в консоль подробные данные сотрудника (имя, код, дату создания и статус активности).

```bash
python main.py --mode info
```
